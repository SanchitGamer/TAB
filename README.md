<div style="text-align: center"><img src="//proxy.spigotmc.org/b1525532d47d299b34e9998b0829235944456989?url=https%3A%2F%2Fi.imgur.com%2FH7zPsMe.png" class="bbCodeImage LbImage" alt="[​IMG]" data-url="https://i.imgur.com/H7zPsMe.png" style="">​</div>TAB is an all-in-one plugin for displaying information in various places, which aims to outperform all similar plugins in terms of features, compatibility and performance.<br>
The compact configuration allows you to get the plugin to work the way you want regardless of how simple or complex you want it to be or how experienced you are. Default configuration already contains useful values and examples to give you a better understanding and get you started quickly with instant results. The extensive wiki answers any and all of your questions.<br>
